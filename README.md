# Отчет по выполнению глав 1-2 учебника по Django

## Основная цель
**Зайти в админку Django на localhost.**
**Цель достигнута.** Административная панель доступна по адресу `http://127.0.0.1:8000/admin/`.

## Выполненные действия

### Глава 1
1.  **Установка Django:** Установлена версия ~4.1 с помощью `pip`.
2.  **Создание проекта:** Создан проект `samplesite` с помощью `django-admin startproject`.
3.  **Создание приложения:** Создано приложение `bboard` с помощью `manage.py startapp`.
4.  **Регистрация приложения:** Приложение `bboard` добавлено в `INSTALLED_APPS` в `settings.py`.
5.  **Запуск сервера:** Отладочный сервер запущен, стартовая страница Django отображается.
6.  **Создание модели:** Объявлена модель `Bb` для объявлений.
7.  **Миграции:** Выполнены команды `makemigrations` и `migrate` для создания таблиц в БД.
8.  **Админка:**
    - Создан суперпользователь (`createsuperuser`).
    - Модель `Bb` зарегистрирована в `admin.py`.
    - Проект русифицирован (изменен `LANGUAGE_CODE`).
    - **Основная цель выполнена:** Вход в `/admin/` осуществлен успешно.

### Глава 2
1.  **Связи между моделями:**
    - Создана модель `Rubric` для рубрик.
    - В модель `Bb` добавлено поле `rubric` (ForeignKey).
    - Выполнены миграции.
2.  **Строковое представление:** Для модели `Rubric` переопределен метод `__str__`.
3.  **URL-параметры:** Реализована страница просмотра объявлений по рубрике (`/bboard/<rubric_id>/`).
4.  **Обратное разрешение URL:** В шаблонах используются теги `{% url %}`.
5.  **Формы:** Создана форма `BbForm`, связанная с моделью `Bb`.
6.  **Контроллеры-классы:** Реализован контроллер `BbCreateView` для добавления новых объявлений.
7.  **Наследование шаблонов:** Создан базовый шаблон `layout/basic.html`.
8.  **Статические файлы:** Добавлены CSS-стили и изображение.
